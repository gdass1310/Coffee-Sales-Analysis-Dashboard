# Coffee Sales Analysis Dashboard (Excel)

## Project Overview

This project focuses on analyzing coffee sales data to understand business performance, customer purchasing behaviour, product profitability, and sales trends.

The objective of this project was to transform raw transactional data into an interactive Excel dashboard that provides meaningful insights into revenue generation, customer behaviour, product performance, and business growth opportunities.

---

# Business Problem

A coffee retail business wanted to analyze its sales data to understand:

- Which coffee products generate the highest revenue?
- Which coffee types and roast categories perform better?
- How do customers contribute to overall sales?
- Which products generate higher profitability?
- What trends can help improve business decisions?

- Product ID

- Customer Information
- Coffee Type
- Roast Type

### Customer Data:
- Customer Details

### Product Data:
- Coffee Type
- Roast Category

# Project Workflow

## 1. Data Preparation & Cleaning

Performed data preparation to make the dataset suitable for analysis:

- Organized raw sales data
- Checked data consistency
- Structured customer, product, and order information
- Prepared data for dashboard creation

---

# 2. Sales Performance Analysis

Analyzed key sales metrics including:

- Total revenue performance
- Sales trends over time
- Quantity sold analysis
- Product-wise sales contribution
- Coffee category performance

---

# 3. Product Analysis

Evaluated product performance based on:

- Coffee type
- Roast type
- Product size
- Pricing
- Profit contribution

Identified products that drive maximum sales and profitability.

---

# 4. Customer Analysis

Analyzed customer behaviour through:

- Customer purchasing patterns
- Loyalty card customers
- Geographic customer distribution
- Revenue contribution by customers

---

# Dashboard Features

Created an interactive Excel dashboard containing:

- Sales KPIs
- Revenue analysis
- Product performance
- Customer insights
- Coffee category comparison
- Profit analysis

Interactive filters were added using Pivot Tables, Pivot Charts, and Slicers for better exploration.

---

# Key Business Insights

The analysis helped identify:

- Best-performing coffee categories and products.
- Customer segments contributing most to revenue.
- Products with higher profitability.
- Sales patterns useful for improving inventory and marketing decisions.

---

# Business Recommendations

Based on the analysis:

- Focus marketing efforts on high-performing coffee categories.
- Promote profitable products to increase margins.
- Use customer purchasing patterns for targeted campaigns.
- Strengthen loyalty programs to improve customer retention.
- Optimize inventory based on product demand trends.
